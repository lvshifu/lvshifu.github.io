使用说明

本程序下载后，先解压缩到任意目录下，然后直接运行，会看到如下界面：

然后，勾选 添加到系统右键菜单 选项
之后在windows文件管理窗口中对执行文件单击鼠标右键会出现如下2个新的选项：

通过 NTLEA 加载运行
通过 NTLEA 设置后加载运行

其中，『通过 NTLEA 加载运行』是使用NTLEA上一次的相同设置启动目标程序。
而『通过 NTLEA 设置后加载运行』选项则是先打开设置界面，待设置完毕之后，点击右下角的『启动』按钮运行目标程序。

通常情况下，设置上只需自行选择右上的『默认区码』下的预设菜单，其他选项默认即可
特别要注意的是『自定区码』属于比较高端的功能，需要一定的知识基础，如果不是对系统代码页非常了解的话请勿使用本功能
而『预设字体』功能则一般用于像light公司的游戏这样有特殊要求的程序

鼠标在选项上悬停几秒，会有此选项的进一步说明信息。
『页码』和『语言』两个输入框在鼠标悬停后还有详细的页码和语言ID列表（但是因为太多而无法显示完整…逃….）

如果在以后的操作中移动了本程序的存放路径位置而造成鼠标右键菜单那两个选项失效的话，只需直接运行主程序，将『添加到系统右键菜单』选项取消一次然后再次勾选即可恢复

本程序是完全绿色软件，若要卸载，只需将『添加到系统右键菜单』选项的勾选取消掉，然后退出并程序即可，不会在系统中留下任何垃圾。

Locale Emulator

(2017/10/09)
Yet Another Region Simulator for modern Windows OS

NTLEA和AppLocale对新系统支持不好，Locale Emulator可以支持Windows 10。

Locale Emulator is a tool similar to MS AppLocale and NTLEA, providing a simulation function that can make an application recognize your OS as in a language other than the real one. It is quite useful when you are trying to play country-specific games.

Locale Emulator is compatible with the following OSes, both 32- and 64-bit:

Windows 7 with Service Pack 1 (Legacy support, see note below)
Windows 8.1 (Legacy support, see note below)
Windows 10
