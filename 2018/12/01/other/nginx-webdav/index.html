Nginx开启Webdav

By lvshifu

发表于 2018-12-01

网上搜到一个开源笔记本应用Joplin，https://joplin.cozic.net/。
开源、跨平台、有移动端、可以Webdav同步、支持Markdown。

Joplin 是一个自由开源的笔记和待办事项应用，可用于 Linux、Windows、macOS、Android 和 iOS。它的主要功能包括端到端加密，Markdown 支持以及通过 NextCloud、Dropbox、OneDrive 或 WebDAV 等第三方服务进行同步。

正好手上有腾讯云服务器，尝试部署了一下。

yum install nginx后，查看配置，默认启用了–with-http_dav_module，但是仍然不能同步。故用源码安装，加入nginx-dav-ext-module模块。

  1. 安装准备
    yum install gcc
    yum install git
    yum install httpd-tools
    wget https://nginx.org/download/nginx-1.14.2.tar.gz

  2. 解压缩
    tar xzvf nginx-1.14.2.tar.gz
    cd nginx-1.14.2/
    cd src

  3. 获取nginx-dav-ext-module
    git clone https://github.com/arut/nginx-dav-ext-module.git

  4. 安装缺失的组件
    yum install pcre-devel
    yum install zlib-devel
    yum install libxml2-devel
    yum install libxslt-devel

  5. 检查用户
    id nginx

  6. configure
    ./configure –with-http_dav_module –add-module=./src/nginx-dav-ext-module/ –group=nginx –user=nginx

  7. 安装
    make &&make install

  8. 更改目录权限
    chown -R nginx:nginx /usr/local/nginx

  9. 验证
    nginx -V

  10. 修改配置
    cd /usr/local/nginx/conf
    vi nginx.conf
    加上include语句
    mkdir vhost
    touch domainname.conf

  11. domainname.conf配置

  12. 生成WEBDAV用户名密码
    htpasswd -c /usr/local/nginx/conf/.htpasswd user1

  13. 启动
    nginx
