frp

https://github.com/fatedier/frp/
frp 是一个高性能的反向代理应用，可以帮助您轻松地进行内网穿透，对外网提供服务，支持 tcp, http, https 等协议类型，并且 web 服务支持根据域名进行路由转发。

ngrok

https://ngrok.com/
老牌穿透工具。可以注册账号免费使用。

Sunny-Ngrok

https://www.ngrok.cc/
提供基于Ngrok和frp两种形式的穿透方式，注册账号即可使用。
速度一般。

sakura-frp

https://www.natfrp.org/
提供多条线路，注册即可使用。使用人数较多，热门线路容易掉线。速度较快。

nat123

http://www.nat123.com/
注册账号后需要实名验证或充值，否则效果不佳。

netapp

https://natapp.cn/
没用过

TeamViewer替代，在TeamViewer弹窗日益频繁的情况下，下面两种方案可以代替

  1. Anydesk
  2. frp+RDP

自己有外网VPS的话，建议自己部署frp或者ngrok，
没有的话，建议使用sakura-frp
